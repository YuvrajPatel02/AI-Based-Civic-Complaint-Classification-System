How it works

  1. Your complaint text is analyzed using TF-IDF + Logistic Regression (NLP).
  2. If you upload a photo, it's analyzed with a CNN / MobileNetV2 image model.
  3. If both are provided, predictions are combined for a more confident result.
  4. A priority level (High / Medium / Low) is assigned using predefined municipal rules.
  5. You receive a unique complaint ID to track your submission on the Dashboard.
